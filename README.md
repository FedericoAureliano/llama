# Prerequisites
- [OCaml](https://ocaml.org/)
- [dune](https://dune.build/)

# Build
```sh
# You can skip this if you want
dune build @install
```

# Run
```sh
# Builds and executes bin/llama.ml
dune exec llama 
```

# Clean
```sh
dune clean 
```
